# Frontend

This frontend is a CRA build served by a small Express server in `server.js`.

## Scripts

- `yarn dev` - local development (CRACO dev server)
- `yarn build` - production build
- `yarn start` - serve the build via `server.js` (binds to `$PORT`)

For deployment steps, see [RAILWAY_DEPLOY.md](../RAILWAY_DEPLOY.md).
